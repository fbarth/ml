Dando continuidade ao post publicado aqui, seguem novos exemplos de clustering, classificação e regras de associação usando o R:

  • Exemplo de regras de associação em uma base de click-stream: este exemplo faz a análise do dataset disponibilizado em  http://fimi.ua.ac.be/data/kosarak.dat. Esta análise utiliza o algoritmo Apriori para criar regras de associação que sumarizam padrões comuns de navegação em um site.
  • Identificação de agrupamentos em mensagens do twitter: este script faz uso do algoritmo Kmeans para identificar grupos de mensagens similares no twitter.
  • Classificação de Spam: os scripts http://rpubs.com/fbarth/classificacaoSpamJ48 e http://rpubs.com/fbarth/classificacaoSpamRandomForest fazem uso do algoritmo J48 e Random Forest, respectivamente, para criar um classificador de Spam para o dataset http://archive.ics.uci.edu/ml/datasets/Spambase.
